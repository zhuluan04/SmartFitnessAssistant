# TabBar 图标说明

TabBar 需要 6 个 PNG 图标文件，建议尺寸 81×81px。

## 所需图标

| 文件名 | 说明 | 颜色 |
|--------|------|------|
| `home.png` | 首页图标（未选中） | 灰色 #636E72 |
| `home-active.png` | 首页图标（选中） | 薄荷绿 #4ECDC4 |
| `recommend.png` | 推荐页图标（未选中） | 灰色 #636E72 |
| `recommend-active.png` | 推荐页图标（选中） | 薄荷绿 #4ECDC4 |
| `profile.png` | 我的页图标（未选中） | 灰色 #636E72 |
| `profile-active.png` | 我的页图标（选中） | 薄荷绿 #4ECDC4 |

## 推荐图标

可以使用以下图标风格：
- 线条风格（Line）
- 扁平填充（Filled）
- 推荐图标库：[IconFont](https://www.iconfont.cn/) 或 [Feather Icons](https://feathericons.com/)

## 临时方案

在图标准备好之前，可以使用 emoji 文字代替，但 TabBar 不支持 emoji，需要自行准备 PNG 图标文件。
