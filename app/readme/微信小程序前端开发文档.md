## 一、项目概述

| 项目 | 内容 |
|---|---|
| 前端形态 | 微信小程序（原生开发） |
| 设计风格 | **清新自然**（薄荷绿 + 白底 + 圆角卡片） |
| 后端服务 | FastAPI（`http://localhost:8000`，已实现） |
| 核心功能 | 用户输入食材文字/上传食物图片 → 后端 Agent 分析 → 渲染 Markdown 营养报告 |
| 导航结构 | 底部三栏 TabBar：**首页** / **推荐** / **我的** |
| 依赖组件 | mp-html（Markdown 渲染） |

### 前后端职责边界

```
微信小程序（前端）              FastAPI（后端）✅ 已实现
┌──────────────────┐           ┌──────────────────────┐
│ · 食材文字输入     │           │ · LangChain Agent     │
│ · 图片上传/压缩    │  HTTP →  │ · 文本模型 qwen3.5-plus│
│ · 推荐食谱展示     │  ← JSON  │ · 视觉模型 qwen-vl-plus│
│ · 健身目标选择     │           │ · Tavily 搜索          │
│ · Markdown 渲染   │           │ · 卡路里计算工具        │
└──────────────────┘           └──────────────────────┘
```

> **原则**：小程序仅做展示和输入壳，所有 AI 逻辑、搜索、卡路里计算全部留在 Python 后端。

## 二、后端 API 接口（已实现 ✅）

> 后端已实现，启动命令：`uv run python main.py api`
> 接口文档地址：`http://localhost:8000/docs`（Swagger UI）

### 2.1 食谱生成接口

**POST `/api/generate_diet`**

#### 请求体

```json
{
  "mode": "text",
  "content": "鸡胸肉150g，西兰花100g，鸡蛋2个",
  "fitness_goal": "减脂",
  "mime_type": "image/jpeg"
}
```

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `mode` | string | 是 | `"text"` 纯文字 / `"image"` 图片 |
| `content` | string | 是 | text：食材文字描述；image：base64 编码（不含 `data:` 前缀） |
| `fitness_goal` | string | 否 | `"减脂"` / `"增肌"` / `"维持"`，默认 `"减脂"` |
| `mime_type` | string | 否 | 图片 MIME 类型，默认 `"image/jpeg"` |

#### 响应体

```json
{
  "success": true,
  "report": "| 排名 | 食谱名称 | 总卡路里 | ... |\n|---|---|---|...",
  "elapsed_seconds": 12.5
}
```

| 字段 | 类型 | 说明 |
|---|---|---|
| `success` | boolean | 请求是否成功 |
| `report` | string | Markdown 格式的营养报告 |
| `elapsed_seconds` | float | Agent 处理耗时（秒） |
| `error` | string | 失败时的错误描述（`success=false`） |

### 2.2 精选推荐食谱接口

**GET `/api/recommend?goal=减脂`**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `goal` | string | 否 | 健身目标：`"减脂"` / `"增肌"` / `"维持"`，默认 `"减脂"` |

#### 响应体

```json
{
  "success": true,
  "recommendations": [
    {
      "id": 1,
      "name": "鸡胸肉西兰花便当",
      "calories": "311 kcal",

      "description": "鸡胸肉切块腌制后滑炒...",
      "nutrition_score": 9,
      "difficulty_score": 8
    }
  ]
}
```

#### 逻辑（index.js）

```javascript
const { request } = require('../../utils/api')

Page({
  data: {
    textInput: '',
    imageBase64: '',
    imagePath: '',
    hasImage: false,
    fitnessGoal: '减脂',
    loading: false,
    goals: [
      { label: '🔥 减脂', value: '减脂' },
      { label: '💪 增肌', value: '增肌' },
      { label: '⚖️ 维持', value: '维持' },
    ],
  },

  onTextInput(e) {
    this.setData({ textInput: e.detail.value })
  },

  onGoalTap(e) {
    this.setData({ fitnessGoal: e.currentTarget.dataset.value })
  },

  chooseImage() {
    wx.chooseMedia({
      count: 1,
      mediaType: ['image'],
      sourceType: ['album', 'camera'],
      success: (res) => {
        const tempPath = res.tempFiles[0].tempFilePath
        const size = res.tempFiles[0].size
        if (size > 4 * 1024 * 1024) {
          wx.showToast({ title: '图片不能超过4MB', icon: 'none' })
          return
        }
        wx.compressImage({
          src: tempPath,
          quality: 70,
          success: (compressed) => {
            const fs = wx.getFileSystemManager()
            const base64 = fs.readFileSync(compressed.tempFilePath, 'base64')
            this.setData({
              imageBase64: base64,
              imagePath: compressed.tempFilePath,
              hasImage: true,
            })
          }
        })
      }
    })
  },

  removeImage() {
    this.setData({ imageBase64: '', imagePath: '', hasImage: false })
  },

  onSubmit() {
    const { textInput, imageBase64, fitnessGoal, hasImage } = this.data
    if (!hasImage && !textInput.trim()) {
      wx.showToast({ title: '请输入食材或上传图片', icon: 'none' })
      return
    }
    this.setData({ loading: true })
    wx.showLoading({ title: 'Agent 思考中...', mask: true })

    request({
      url: '/api/generate_diet',
      method: 'POST',
      data: {
        mode: hasImage ? 'image' : 'text',
        content: hasImage ? imageBase64 : textInput,
        fitness_goal: fitnessGoal,
      },
    }).then((res) => {
      wx.hideLoading()
      if (res.success) {
        wx.setStorageSync('currentReport', res.report)
        wx.navigateTo({
          url: `/pages/report/report?time=${res.elapsed_seconds}`
        })
      } else {
        wx.showModal({ title: '分析失败', content: res.error, showCancel: false })
      }
    }).catch((err) => {
      wx.hideLoading()
      wx.showModal({ title: '网络错误', content: '请检查后端服务', showCancel: false })
    }).finally(() => {
      this.setData({ loading: false })
    })
  },
})
```

const { request } = require('../../utils/api')

Page({
  data: {

const app = getApp()

function request(options) {
  return new Promise((resolve, reject) => {
    wx.request({
      url: app.globalData.apiBaseUrl + options.url,
      method: options.method || 'GET',
      data: options.data,
      timeout: app.globalData.requestTimeout,
      header: { 'Content-Type': 'application/json' },
      success: (res) => {
        if (res.statusCode === 200) resolve(res.data)
        else reject(new Error(`HTTP ${res.statusCode}`))
      },
      fail: (err) => reject(err),
    })
  })
}

module.exports = { request }

## 七、关键注意事项

### 7.1 请求超时

后端 Agent 单次请求耗时 **10-30 秒**。

| 配置项 | 推荐值 | 说明 |
|---|---|---|
| `wx.request` timeout | `60000`（60秒） | 切勿使用默认值 |
| loading 动画 | 必须有 | `wx.showLoading` + 分阶段文案 |
| 超时兜底 | 重试一次 | 弹窗提示"分析超时，是否重试" |

### 7.2 图片处理

| 限制项 | 说明 |
|---|---|
| 单张上限 | 10MB（需压缩后发送） |
| base64 传输 | < 2MB（压缩后） |
| 支持格式 | jpg、png、webp |
| 压缩质量 | `wx.compressImage` quality=70 |
| 编码方式 | `readFileSync(path, 'base64')`，**不含** `data:` 前缀 |

### 7.3 域名白名单

- **开发阶段**：微信开发者工具 → 勾选「不校验合法域名」
- **生产阶段**：必须配置 HTTPS，在微信公众平台添加 request 合法域名

### 7.4 安全

- 小程序端**不存储** API Key
- 图片仅用于本次分析，后端不持久化
- 生产环境需 HTTPS + 请求鉴权

### 7.5 数据传递

- 报告内容较长，使用 `wx.setStorageSync` 传递，**不要**用 URL 参数
- 健身目标使用 `wx.getStorageSync` / `wx.setStorageSync` 跨页面共享
